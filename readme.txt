vimagee
==================

A simplistic image viewer.

Image loading is done via DevIL (http://openil.sourceforge.net/about.php) which means the following formats are supported:
.bmp, .cut, .dds, .doom, .exr, .hdr, .gif, .ico, .jp2, .jpg,.lbm, .mdl, .mng, .pal, .pbm, .pcd, .pcx, .pgm, .pic, .png, .ppm, .psd, .psp, .raw, .sgi, .tga and .tif files.
