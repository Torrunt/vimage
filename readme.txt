vimage
===============================

A simplistic image viewer.

Image loading is done via DevIL (http://openil.sourceforge.net/about.php) which means the following formats are supported:
.bmp, .cut, .dds, .doom, .exr, .hdr, .gif, .ico, .jp2, .jpg,.lbm, .mdl, .mng, .pal, .pbm, .pcd, .pcx, .pgm, .pic, .png, .ppm, .psd, .psp, .raw, .sgi, .tga and .tif files.

Controls
--------
Left-Click to Drag
Right-Click to Close
Scroll Wheel to Zoom - Hold SHIFT to zoom faster, Hold CTRL to zoom based on center (doesn't look very smooth at the moment though)
Middle-Click to go between actual size and monitor height
Left/Right Arrows (or Page Up/Down) to go through images in a folder
Up/Down Arrows to Rotate
F to Flip Horizontally
S to Toggle Smoothing
T to Toggle a light grey background for images with transparency
Space to pause Animated GIFs
,/. to go through Animated GIF frames
